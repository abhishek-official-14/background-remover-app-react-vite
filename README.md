# AI Background Remover

Professional AI-powered background removal web application built with React + Vite.

## Features

- 🚀 **Instant Background Removal** - Remove backgrounds from images in seconds
- 🎯 **Precise Edge Detection** - Accurate hair and fur segmentation
- 📦 **Batch Processing** - Process multiple images simultaneously
- 🎨 **Background Editor** - Change backgrounds, add effects, and more
- 🔒 **Privacy First** - Images are automatically deleted after processing
- 🌙 **Dark Mode** - Beautiful dark/light theme support
- 📱 **Fully Responsive** - Works perfectly on all devices
